
# Basel IV Dashboard

## How to Run Locally
1. Install dependencies:
```
pip install -r requirements.txt
```
2. Run the app:
```
streamlit run test1.py
```

## How to Deploy on Streamlit Cloud
1. Push `test1.py` and `requirements.txt` to a GitHub repo.
2. Go to [Streamlit Cloud](https://share.streamlit.io/) and create a new app.
3. Select your repo, branch, and file path (`test1.py`).
4. Deploy and share the link.
